### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/instatrack-pro.git
   cd instatrack-pro
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Build the project**:
   ```bash
   npm run build
   ```

4. **Load into Chrome**:
   - Open Chrome and navigate to `chrome://extensions/`.
   - Enable **Developer Mode** (top right toggle).
   - Click **Load unpacked** and select the `dist` folder generated by the build.

## ğŸ“– How to Use

1. **Login**: Ensure you are logged into Instagram in your current Chrome session.
2. **Open Extension**: Click the **InstaTrack Pro** icon in your toolbar.
3. **Scan**: Click "Start Scan". You will see your profile ID and CSRF token detected automatically.
4. **Relax**: You can close the popup or switch tabs. A notification will appear when the scan is finished.
